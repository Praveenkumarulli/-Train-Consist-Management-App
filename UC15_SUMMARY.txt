================================================================================
                           KEY CONCEPTS TAUGHT
================================================================================

1. Custom Runtime Exceptions
   - CargoSafetyException extends RuntimeException
   - Unchecked exceptions don't require explicit throws
   - Thrown during program execution

2. try-catch-finally Pattern
   - try: Code that might throw exceptions
   - catch: Handle exceptions gracefully
   - finally: ALWAYS execute (cleanup/logging)

3. Graceful Failure Handling
   - Prevent application crashes
   - Provide error feedback to users
   - Continue execution after failures

4. State Management
   - Revert state on exception
   - Maintain consistency
   - Keep system in safe state

5. Domain-Specific Safety Rules
   - Petroleum + Rectangular bogie = UNSAFE
   - Petroleum + Cylindrical bogie = SAFE
   - Enforced through exception handling
